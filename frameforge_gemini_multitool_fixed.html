10 araç, tek arayüz

FrameClone, PoseClone, LightingClone, CameraMatch, LocationSwap, OutfitTransfer, SceneBuilder, Influencer Generator, Reference Mixer ve AI Photoshoot Simulator tek panelde birleşti. Gemini API ile görselleri ve yazılı brief'leri analiz eder, ardından DALL·E, Midjourney ve Flux için ayrı ayrı daha stabil prompt paketleri üretir.

Nasıl çalışır
Gemini API key gir → modül seç → görsel/prompt yükle → Gemini structured JSON oluştursun → sana kullanıma hazır prompt seti dönsün.
Çıktılar
Scene analysis, subject transfer rules, ana prompt, negative prompt, DALL·E / Midjourney / Flux varyantları ve üretim notları.
